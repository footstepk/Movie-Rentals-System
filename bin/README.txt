/**
Testing note for MovieRentalSystem.
*/

Press 1: Searching a title.
When prompted, enter some value i.e. classic or cla
Result will be displayed, else
follow the further instruction.

Press 2: Searching a customer.
When prompted, enter some value i.e. 900009 or <ENTER> without values or 9
Result will be displayed, else
follow the further instruction.

Press 3: creating a title
When prompted, enter some value
Result will be displayed, else
follow the further instruction.

Press 4: registering a customer.
When prompted, enter some value
Result will be displayed, else
follow the further instruction.

Press 5: updating a customer.
When prompted, enter some value
Result will be displayed, else
follow the further instruction.

Press 6: registering a customer rented title.
When prompted, enter some value i.e. 900001 to be validated.
Result will be displayed, else
follow the further instruction.
